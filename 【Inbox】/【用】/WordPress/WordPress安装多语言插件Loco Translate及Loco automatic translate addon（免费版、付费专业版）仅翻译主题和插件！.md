**Loco Translate是针对主题&插件进行翻译（不同于其它翻译插件！）**

  
支持自动翻译功能，有串接 DeepL、Google、Microsoft 及 Yandex 这几个知名的翻译 API  
但如果你使用 Loco Translate，还是有些文字无法翻译成功，你也可以通过 Poedit 软件翻译，它可以抓取更多主题或插件内的文字，进而实现 100% 系统字翻译。
